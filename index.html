About the KCGB Interactive Map

The Keep Coral Gables Beautiful program (KCGB) is an affiliate program of Keep America Beautiful for the City of Coral Gables. This web map application is a tool to visualize and manage the city's beautification efforts. All events and programs related to this initiative are accessible through this platform.

How to Use This Application

  • Explore the Map: The main map displays all event and program locations for KCGB.
  • Search: Use the search bar to find a specific address.
  • Events/Programs Information: Click on an icon on the map to see its name. You will also see relevant details about the event/program and an option to view more information.
  • Sidebar: The "Events and Programs" sidebar provides a list of all events and programs by category. You can click on any item to see more details about specific KCGB initiatives.
  • Tutorial: For a guided tour, click the "START TUTORIAL" button in the navigation bar. Please note that this feature is not available on mobile devices.
